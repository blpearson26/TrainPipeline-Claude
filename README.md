# Training Management Application

A comprehensive system for managing AI and Product Management training pipelines.

## Features
- Track trainings through 7 pipeline stages
- Client and training management
- Financial tracking
- Session scheduling
- Analytics dashboard
- Persistent data storage

## Technologies
- React
- Vite
- Tailwind CSS v4
- Lucide React Icons

## Development
```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Deployment

This app automatically deploys to GitHub Pages on every push to main branch.
